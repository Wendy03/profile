Vue 電商網站：11街咖啡

Demo | Repo

創作理念：

11 街小小庭院是家庭聚會烘豆地方，烘豆技術來自 90 多歲師傅，從旁邊觀摩到自己獨立作業。庭院一顆咖啡樹，曾經有一年產量不錯，參與從採收、水洗、日曬、挑豆、烘豆、磨粉、製作耳掛式咖啡，分送給親朋好友。專題以最熟悉咖啡豆為主題。

使用技術：

  1. Vue 3 Options API + Vite 建立與打包專案
  2. Vue Router 建構路由
  3. 串接 RESTful API
  4. Bootstrap 5 + SCSS 規劃響應式版型
