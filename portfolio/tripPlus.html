協作專案: 旅遊募資平台 （負責後端）

Demo | 前端-前台 | 前端-後台 | 後端

創作理念：

旅行募資平台-通過網絡平台連接旅行者和支持者，讓旅行者能夠籌集到旅行資金， 實現自己的旅行夢想，同時讓支持者可以享受旅行者的旅行故事和體驗。

使用技術：

  • 設計：Miro
  • 前端：Next.js、TaiwindCSS、Chakra UI、Typescript
  • 後端：Node.js、Express、Mongo Atlas、Mongoose

負責後端

  • 顧客中心 API
  • 後台新增商品 API
  • Socket.IO
